# 🔴 FlaNews - Portal de Notícias

O **FlaNews** é um projeto de portal de notícias focado em entregar uma experiência de leitura limpa e responsiva. Este projeto foi desenvolvido para consolidar conhecimentos em tecnologias fundamentais da web (HTML, CSS e JS) e princípios de UX/UI.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estruturação semântica para melhor acessibilidade e SEO.
- **CSS3:** Estilização moderna utilizando **Flexbox** e **Grid Layout** para garantir responsividade.
- **JavaScript (ES6+):** Lógica de interatividade e manipulação dinâmica do DOM.

## 📱 Diferenciais do Projeto

- **Design Mobile-First:** Interface totalmente adaptável para dispositivos móveis, tablets e desktops.
- **Identidade Visual:** Aplicação de cores e tipografia focadas na experiência do usuário e na temática do projeto.
- **Performance:** Código leve e otimizado para carregamento rápido.
