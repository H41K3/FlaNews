# 🔴⚫ FlaNews

O **FlaNews** é um portal dinâmico e interativo focado em notícias, calendário de jogos e estatísticas do Clube de Regatas do Flamengo.

Este projeto foi desenvolvido e refatorado com o objetivo de aplicar conceitos modernos de **Desenvolvimento Web Front-End** (HTML, CSS e JavaScript Vanilla), elevando uma estrutura básica para um nível profissional, com forte foco em manipulação do DOM, código limpo e UI/UX design.

## 🚀 Funcionalidades

- **Dashboard Dinâmico:** Página inicial estilo "vitrine" com resumo das últimas notícias, card do próximo confronto e classificação atualizada do G4.
- **Tabela do Brasileirão Profissional:** Tabela de classificação 100% gerada dinamicamente via JavaScript. Inclui indicador de "Últimos Jogos" (estilo dashboard esportivo) com indicativos visuais de vitórias (verde), empates (cinza) e derrotas (vermelho).
- **Cards de Jogos:** Seção de "Próximos Jogos" com design em cards modernos, destacando as competições com cores específicas no cabeçalho (Libertadores, Brasileirão, Copa do Brasil).
- **Notícias Interativas (Acordeão):** Grid de notícias responsivo. O usuário pode clicar em "Ler matéria completa" para expandir o texto suavemente na mesma tela (sem recarregar a página), através de manipulação de classes CSS pelo JS.
- **Busca Inteligente:** Barra de pesquisa que redireciona o usuário rapidamente para as abas corretas do site com base em palavras-chave.

## 💻 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica rigorosa (`<header>`, `<main>`, `<section>`, `<nav>`, `<footer>`).
- **CSS3:** - Layouts avançados utilizando **CSS Grid** e **Flexbox**.
  - UI Premium: Efeitos de *Hover* flutuantes, *box-shadow* profundas, e transições suaves (*transitions*).
  - *Zebra Striping* dinâmico nas tabelas para facilitar a leitura.
- **JavaScript (ES6+):** - Renderização dinâmica do DOM (Criação de elementos HTML a partir de Arrays de objetos com dados reais).
  - *Event Listeners* avançados para navegação, interação de botões (Acordeão) e busca.

## 🛠️ Como executar o projeto localmente

Não é necessária nenhuma instalação complexa. O projeto roda nativamente no navegador.

1. Crie uma pasta, acesse-a pelo terminal e clone este repositório:

   git clone <https://github.com/H41K3/FlaNews.git>

2. Abra a pasta do projeto no seu editor de código favorito (ex: VS Code).

3. Abra o arquivo index.html diretamente no seu navegador de preferência, ou utilize a extensão Live Server para visualização em tempo real.
